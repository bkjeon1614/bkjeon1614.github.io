#### OpenClaw 란
로컬에서 실행되는 Personal AI Agent로 AI와(chatgpt, claude, gemini 등) 연동하여 Telegram, Slack, Discord 등 여러 채널을 통해 명령을 받고, 브라우저 제어/파일관리/이메일작성 등 `실제 작업을 자율적으로 수행`하는 프로그램입니다.     
     
     
#### OpenClaw 설치에 필요한 프로그램 설치
```
// git 설치
$ sudo apt update && sudo apt install curl build-essential -y
$ sudo add-apt-repository ppa:git-core/ppa -y
$ sudo apt update
$ sudo apt install git -y

// nodejs 설치
$ curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.40.1/install.sh | bash
$ source ~/.bashrc
$ nvm install node
```
